## HOW TO USE THIS PACKAGE

### Deployment Path 1: Claude Projects (Chat Mode)
1. Create new Claude Project named "Research Hub"
2. Upload docs to knowledge base: CLAUDE.md, OPERATING_RULES.md, RESEARCH_MODE_GUIDES.md, QUESTIONNAIRE_TEMPLATES.md
3. Copy CLAUDE.md → Project custom instructions
4. Add conversation starters (from SETUP_GUIDE.md)
5. Done. Users now have access to all research modes.

### Deployment Path 2: Cowork (Multi-Agent Mode)
1. Copy skill files from `/new-skills/` to `~/.claude/skills/`
2. Use SETUP_GUIDE.md → Cowork mode section
3. Spawn agents per research track
4. Coordinate with task-coordination-strategies skill
5. Done. Parallel research is live.

### Deployment Path 3: Share as Standalone Package
1. Give user this zip file
2. User reads INDEX_AND_QUICKSTART.md first
3. User picks deployment path (Chat or Cowork)
4. User follows SETUP_GUIDE.md
5. Done. User is researching.
